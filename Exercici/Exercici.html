Exercicis amb JQuery

Exercici 0

  1. Quan l'usuari prem el botó amb id boto, validar entrada.
  2. En cas de ser validar, mostrar el nom del dia de la setmana corresponen al número dins l'etiqueta amb id res
  3. En cas de no ser valida, netejar la caixa de text i mostra el valor indicat i l'avis dins l'etiqueta amb id res d'aquesta pàgina
Exercici

Exercici 1

Fent servir la funció .height(), mostrar l'altura de totes les etiquetes filles del body dins l'etiqueta amb id res1, després fer que cada etiqueta p tingui una altura de 80px, excepte aquelles que tinguin una imatge a dins seu.

Exercici 2

Fer servir la funció .outerheight() per recuperar els valors de totes les eitquetes p i comparar-les amb els valors obtinguts en l'exercici anterior. Mostrar els valors dins l'etiqueta amb id res2

Exercici 3

Afegir dos botons abans d'aquesta imatge. Els botons s'han de dir "ocultar" i "mostrar" i han de permetre amagar i mostrar la imatge, respectivament, fent ús de hide i show.

Exercici 4

Afegir un select després de la imatge, amb les opcions que hi ha dins l'array opcions. Quan es seleccioni alguna opció es modificarà l'estat de la imatge. Ajuda: slideDown,slideUp,fadeIn,fadeOut,fadeTo.
